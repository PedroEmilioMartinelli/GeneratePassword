# Gerador de Senhas Aleatórias em Kotlin

### Descrição
Este projeto contém uma função em Kotlin que gera uma senha aleatória com base em um comprimento especificado pelo usuário. A senha gerada inclui letras maiúsculas, letras minúsculas, números e caracteres especiais.

### Funcionalidades
Geração de Senhas: A função generatePassword aceita um parâmetro de comprimento e gera uma senha aleatória com o comprimento especificado.
-  Tipos de Caracteres Incluídos:
-  Letras maiúsculas (A-Z)
-  Letras minúsculas (a-z)
-  Números (0-9)
-  Caracteres especiais (!, @, #, $, %, ¨, &, *, (, ))
